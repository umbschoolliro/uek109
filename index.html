Befehle um ein Github auf der VM einzurichten:
1. Zuerst muss man "sudo apt install git" machen
(2. Dann kann man noch den SSH-Server installieren wenn man will mit "sudo apt install openssh-server")
2. Danach erstellt man im Browser ein Repository.
3. Man muss noch einen Token erstellen. Im Github unter Settings -> Developer Settings -> Personal Access Token -> Token (classic) -> Generate new token (classic). Diesen Token muss man umbedingt speichern.
4. Jetzt kann man das Repository mit dem Befehl "git clone https://github.com/umbschoolliro/uek109.git" runterclonen
5. WÃ¤hrend diesem befehl wird man nach Benutzernamen und Passwort(Token) gefragt. Diese dann eingeben. 6. Jetzt hat man das Repository heruntergeladen und kann es lokal benutzen. 7. Wenn man das File jetzt hochladen will macht man zuerst "git add index.html"
8. Dann kann man "git commit -m 'Index erstellt'" benutzen um einen Commit zu erstellen. 9. Als letzten Schritt braucht man "git push" damit das File hochgeladen wird. Da muss man auch wieder Username und Token eingeben.
